# M01 - Lista - Exercício 21 - Quadrado dos Ímpares

## Objetivo

Mostrar o quadrado de todos os números ímpares, de 1 até 200.

## Instruções

1. Escreva um algoritmo em Dart que apresente o quadrado de todos os números ímpares, de 1 até 200.

```

  1 * 1 == 1
  3 * 3 == 9
  5 * 5 == 25
  7 * 7 == 49
  9 * 9 == 81
  11 * 11 == 121
  ...
  23 * 23 == 529
  ...
  75 * 75 == 5625
  ...
  119 * 119 == 14161
  121 * 121 == 14641
  123 * 123 == 15129
  125 * 125 == 15625
```
